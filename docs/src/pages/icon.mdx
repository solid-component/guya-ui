# Icon 图标
happy-ui 提供了一套常用的图标集合。

# 安装
## 使用包管理器

```bash
# 选择一个你喜欢的包管理器

# NPM
$ npm install solid-dada-icons
# Yarn
$ yarn add solid-dada-icons
# pnpm
$ pnpm install solid-dada-icons
```

# 图标集合
只要你安装了solid-dada-icons, 就可以在任意版本里使用 SVG 图标。
[更多](https://element-plus.gitee.io/zh-CN/component/icon.html#icon-collection)
